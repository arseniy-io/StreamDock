# Локальные модели StreamDock

StreamDock не включает модели распознавания речи в Git-репозиторий. Крупные модели занимают до нескольких гигабайт, поэтому пользователь загружает только нужные варианты.

## Как модели загружаются

- автоматически при первом использовании выбранного режима;
- заранее через `download_models.bat`; без аргументов он загружает GigaAM, Whisper `large-v3` и модели разделения спикеров;
- один раз: после загрузки файлы используются из локального кеша;
- без облачной обработки: аудио не отправляется модели через внешний API.

Для первой загрузки нужен интернет. После этого локальный файл можно транскрибировать без сети.

## Предварительная загрузка

Обычный двойной щелчок по `download_models.bat` загружает рекомендуемый набор: GigaAM, Whisper `large-v3` и модели разделения спикеров. Загрузка занимает примерно 3,4 ГБ, а для безопасного завершения лучше иметь не меньше 4,5 ГБ свободного места.

Дополнительные варианты запуска из PowerShell:

```powershell
# Только быстрая русская модель GigaAM
.\download_models.bat --gigaam-only

# GigaAM и более компактный Whisper small, без моделей спикеров
.\download_models.bat --whisper-model small --skip-speaker-models

# Только Whisper small
.\download_models.bat --whisper-model small --skip-gigaam --skip-speaker-models

# Рекомендуемый набор без GigaAM
.\download_models.bat --skip-gigaam
```

Параметр `--whisper-model` принимает `tiny`, `base`, `small`, `medium` или `large-v3`. Его можно повторить, чтобы подготовить несколько моделей.

Предварительная загрузка убирает ожидание скачивания перед первой задачей, но не ускоряет само распознавание. Whisper `large-v3` после загрузки всё равно требует больше времени и ресурсов, чем компактные модели.

## Режимы

### Совместный режим

Предназначен прежде всего для русскоязычных технических видео. GigaAM распознаёт всю речь, затем Whisper `large-v3` проверяет только подозрительные фрагменты с терминами, числами и латиницей.

Преимущество режима - более аккуратные технические места при меньшем времени, чем у полного запуска `large-v3`. Результат всё равно зависит от качества звука, дикции и тематики.

### GigaAM

Локальная модель для русской речи. Это основной быстрый вариант для лекций, интервью и разговорных видео на русском. В текущей сборке работает через CPU.

### Whisper

Универсальный вариант для русского, английского и других языков.

| Модель | Когда выбирать | Нагрузка |
| --- | --- | --- |
| `tiny` | Черновик, быстрый поиск по записи | Минимальная |
| `base` | Быстрая транскрибация с небольшим улучшением качества | Низкая |
| `small` | Повседневный баланс качества и скорости | Средняя |
| `medium` | Более сложная речь, если доступно больше времени и памяти | Высокая |
| `large-v3` | Максимальное качество Whisper важнее скорости | Самая высокая |

Whisper автоматически использует совместимую NVIDIA CUDA, если она доступна. В остальных случаях модель работает на CPU.

## Готовые субтитры

При источнике текста «Сначала готовые субтитры» приложение действует так:

1. ищет подходящие ручные субтитры;
2. затем проверяет автоматические субтитры;
3. если подходящего текста нет, запускает выбранную локальную модель.

Поэтому наличие модели не всегда означает, что она будет запущена для каждого видео.

## Разделение по спикерам

Разделение спикеров использует отдельные локальные модели. Функция по умолчанию выключена. Если модели не были подготовлены через `download_models.bat`, они загрузятся при первом включении.

Можно оставить автоматическое определение или указать от 2 до 10 спикеров. Заданное число обычно помогает на интервью и вебинарах, если состав участников известен заранее.

Разделение не изменяет содержание транскрибации, а добавляет к сегментам подписи `Спикер 1`, `Спикер 2` и так далее. На музыке, шуме, очень коротких репликах и одновременной речи возможны ошибки.

## Ориентировочный объём

Фактический размер зависит от версии и внутреннего формата модели.

- GigaAM вместе с детектором речи - несколько сотен мегабайт;
- небольшие Whisper - от десятков до нескольких сотен мегабайт;
- средние Whisper - сотни мегабайт или больше;
- Whisper `large-v3` - несколько гигабайт;
- модели разделения спикеров - дополнительный объём порядка десятков мегабайт.

Для `large-v3` оставляйте несколько гигабайт свободного места сверх размера модели: приложению также нужны временные файлы и место под результаты.

## Где хранятся файлы

- Whisper - `models`;
- GigaAM и детектор речи - `models/huggingface`;
- разделение спикеров - `models/speaker-diarization`.

Папка `models` исключена из Git. Не добавляйте модели в коммиты и архив исходного кода. Повторный запуск загрузчика проверяет кеш и не скачивает готовые файлы заново.

## Как удалить модели

Остановите StreamDock и удалите ненужные файлы из папки `models`. При следующем выборе удалённой модели приложение загрузит её снова.

Если необходимо сохранить модели перед обновлением или переносом проекта, скопируйте папку `models` целиком.

## Частые ошибки

### Не хватает оперативной или видеопамяти

Выберите модель Whisper меньшего размера, отключите разделение спикеров или используйте GigaAM для русской речи. Закройте другие тяжёлые приложения перед повторным запуском.

### Загрузка прервалась

Проверьте соединение и свободное место, затем повторите операцию. Уже полученные файлы кеша обычно используются повторно.

### CUDA найдена, но Whisper запускается на CPU

Причиной может быть несовместимость драйвера или локальных библиотек CUDA. Обновите драйвер NVIDIA. CPU остаётся поддерживаемым запасным режимом и не требует CUDA.
