# StreamDock для Chrome

Расширение находит медиапоток на открытой странице и передаёт его локальному приложению на `http://127.0.0.1:8765`. Видео и данные доступа не отправляются во внешние сервисы.

## Установка

1. Один раз запустите `install.bat` в корне проекта.
2. Откройте в Chrome страницу `chrome://extensions`.
3. Включите «Режим разработчика» в правом верхнем углу.
4. Нажмите «Загрузить распакованное расширение».
5. Выберите папку `browser-extension`.
6. Закрепите StreamDock на панели Chrome.

После изменения файлов расширения нажимайте круглую кнопку перезагрузки на его карточке в `chrome://extensions`.

## Использование

1. Откройте страницу с доступным вам видео и включите воспроизведение на несколько секунд.
2. Нажмите значок StreamDock.
3. Если приложение остановлено, нажмите «Запустить».
4. Если список пуст, нажмите кнопку обновления.
5. Выберите первый рекомендуемый поток «Видео + звук» и нажмите «Скачать в хорошем качестве».
6. В открывшейся отдельной вкладке следите за загрузкой. Маленькое окно расширения теперь можно закрыть.
7. После работы откройте расширение и нажмите «Остановить».

Если страница отдаёт изображение и звук отдельными дорожками, StreamDock ставит общий `master.m3u8` первым и скрывает технические однодорожечные варианты. Локальное приложение выбирает лучшее качество, объединяет дорожки и проверяет итоговый файл перед сохранением.

Кнопка «Остановить» доступна при любом подтверждённом запуске StreamDock, в том числе после ручного запуска или перезапуска расширения. Backend выдаёт локальному помощнику временный секрет текущего процесса, поэтому команда не затрагивает другие проекты Python и посторонние процессы.

Прямой эфир записывается с момента подключения к найденному потоку. Потоки с DRM-защитой не поддерживаются и не обходятся.

Расширению нужен доступ к страницам, чтобы заметить адрес видеопотока. URL, cookies и заголовки авторизации хранятся только в памяти текущего сеанса Chrome и удаляются после передачи локальному приложению. В постоянном хранилище остаются только название, номер задачи и безопасный снимок прогресса последних десяти загрузок.
