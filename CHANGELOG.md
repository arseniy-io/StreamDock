# История изменений

Все заметные изменения StreamDock фиксируются в этом файле.

## 0.4.0 - первый публичный beta-релиз

- скачивание видео и аудио через yt-dlp с локальной обработкой FFmpeg;
- транскрибация через Whisper, GigaAM и совместный режим;
- быстрый результат из готовых субтитров;
- Markdown, TXT, SRT и VTT;
- очередь ссылок, диапазон транскрибации и компактный редактор;
- технический словарь для IT, DevOps и тем об искусственном интеллекте;
- необязательное локальное разделение по спикерам;
- загрузка и транскрибация локальных медиафайлов;
- расширение Chrome для доступных пользователю страниц;
- безопасный запуск и остановка StreamDock из расширения.
